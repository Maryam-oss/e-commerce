# Responsive E-Commerce Website

A responsive front-end e-commerce web application built using **HTML5**, **CSS3**, and **Bootstrap 5**.  
This project demonstrates a complete e-commerce user flow with clean UI design, responsive layouts, and multiple pages, without using any backend or database.


## Project Overview

The Responsive E-Commerce Website is designed to simulate a real-world online shopping platform.  
It focuses on front-end development concepts such as layout structuring, responsiveness, reusable UI components, and user navigation flow.

All functionalities are **UI-based only**, as the project does not include backend logic or database integration.


## Technologies Used

- HTML5  
- CSS3  
- Bootstrap 5  
- Basic JavaScript (for UI toggling only)

## Pages Implemented

- **Home Page** – Landing page with navigation  
- **Products Page** – Displays all available products using Bootstrap cards  
- **Product Detail Pages** – Individual detail page for each product  
- **Cart Page** – Shows selected products with quantity and total (UI only)  
- **Login / Signup Page** – Toggle-based authentication UI  
- **Admin Dashboard** – Admin panel UI with sidebar, stats, and table  


## Features

- Fully responsive design (Mobile, Tablet, Desktop)
- Clean and consistent Bootstrap navbar across all pages
- Product listing with images, price, and view buttons
- Individual product detail pages
- Shopping cart UI with quantity controls and total summary
- Login & Signup toggle using simple JavaScript
- Admin dashboard UI for management demonstration
- Organized folder structure and clean code



## Important Note

This project is **front-end only**.  
All cart, login, and admin functionalities are implemented as **UI demonstrations** without backend processing.


## Conclusion

This project demonstrates a complete front-end e-commerce structure with a focus on responsiveness, UI/UX design, and clean coding practices using HTML, CSS, and Bootstrap.
